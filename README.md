# vishnus17.github.io

Personal portfolio of **Vishnu S** — Cloud &amp; DevOps Engineer, AWS Community Builder, LiFT'22 Scholar.

Live at **[vishnus17.github.io](https://vishnus17.github.io)**.

## Stack

Plain HTML, CSS, and vanilla JS. No build step — served directly by GitHub Pages.

- `index.html` — single-page portfolio
- `css/portfolio.css` — theme, layout, animations
- `js/portfolio.js` — typed hero, scroll reveal, mobile nav
- `images/` — project screenshots and article thumbnails
